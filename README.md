## 📌 About The Project

Agentathon Web is a modern event website designed to showcase:

- 🤖 Agentic AI Hackathon
- 🧠 AI Workshops & Speaker Sessions
- 📅 Event Schedule
- 🎤 Industry Mentors & Speakers
- 🏆 Competitions & Team Activities
- 📢 Registrations and Announcements

The platform is built with a futuristic UI inspired by AI and cyberpunk aesthetics to create an immersive experience for participants.

## ✨ Features

- Responsive modern UI
- Animated AI themed design
- Interactive sections
- Event information & schedules
- Speaker showcase
- Registration integration
- Fast deployment using Vercel
